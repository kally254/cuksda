# CUKSDA Website

A modern, fully responsive website built with HTML, CSS, and JavaScript featuring smooth animations, mobile-friendly design, and comprehensive sections.

## Features

✨ **Responsive Design**
- Mobile-first approach
- Optimized for all screen sizes
- Smooth transitions and animations

📱 **Mobile Menu**
- Hamburger menu icon for mobile devices
- Smooth open/close animations
- Click outside to close functionality

🎨 **Hero Sections**
- Eye-catching hero sections on all pages
- High-quality background images
- Call-to-action buttons

📄 **Pages**
1. **Home** - Welcome page with features overview
2. **About** - Company mission, vision, and team information
3. **Services** - Detailed service offerings
4. **Contact** - Contact form and company information

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #007bff;
    --secondary-color: #6c757d;
    --dark-color: #212529;
    --light-color: #f8f9fa;
}
```

### Images
Replace the image URLs in HTML files with your own images:
- Hero background images
- Team member photos
- Service icons

### Content
Update text content in respective HTML files to match your brand.
